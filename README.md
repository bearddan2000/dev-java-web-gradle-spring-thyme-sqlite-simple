# dev-java-web-gradle-spring-thyme-sqlite-simple

## Description
A thyme springboot java gradle build,
that connects to in memory sqlite database.

## Tech stack
- springboot
  - thyme
- gradle
  - sqlite drivers
  - PostConstruct annotation
- bootstrap
- jquery
- dataTable

## Docker stack
- gradle:7.6.4-jdk17-alpine

## To run
`sudo ./install.sh -u`
Available http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
